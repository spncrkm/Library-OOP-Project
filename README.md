# Library-UI-OOP-proj

## Overview

This project is a Library Management System that allows users to manage books, authors, genres, and user information. This project also includes functionality for checking overdue books.

## Features

- **Book Operations:**
- Add new books into library
- Borrow books
- Return books
- Search for books
- Display all books

- **User Operations:**
- Add new users
- View user details
- Display all users

- **Author Operations:**
- Add new authors
- View author details
- Display all authors

- **Genre Operations:**
- Add new genre
- View genre details
- Display all genres

- **Overdue Books:**
- Check for overdue books
